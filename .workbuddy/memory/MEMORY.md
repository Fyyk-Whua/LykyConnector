# 项目长期记忆 — 邻医云 B2C 对接程序

## 项目概况
- 为药店/ISV 开发邻医云 B2C 开放平台的本地桌面对接程序（C# .NET 8 + WPF）。
- 核心诉求：桌面界面 + 进程保护 + 防随意退出 + 开机自启，保证 7×24 稳定运行医药订单同步。

## 接口要点
- 平台：邻医云 B2C 开放平台（邻医快药），store-api.lyky.cn
- 14 个接口：7 主动调用（库存/货位/价格/批号/电子面单获取取消/发货同步）+ 7 推送回调（订单/售后/发货/代发/退货/出库/追溯码）
- 签名：系统参数+app_secret 排序拼接 → MD5 → Base64
- 数据格式：POST, application/x-www-form-urlencoded
- 医药特有：处方单、追溯码、批准文号、收货信息加密

## 技术决策
- C# .NET 8 + WPF（MVVM）+ Kestrel 内嵌 HTTP 收推送 + SQLite 持久化队列 + Serilog + 看门狗双进程互守
- 不选 Electron（内存高、进程保护弱）/ Python（打包与守护复杂）
